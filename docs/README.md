# Project Emporion

An online auction platform built with modern technologies focusing on security, scalability, and user experience.

## Overview
Project Emporion is a comprehensive auction system designed for the Nigerian market with plans for broader expansion. The platform implements industry best practices for security, payments, and real-time interactions.

## Key Features
- Secure user authentication and verification
- Real-time auction bidding with anti-sniping measures
- Integrated payment processing (Paystack, Flutterwave)
- Escrow service for secure transactions
- Auction-linked chat for buyer-seller communication
- Comprehensive analytics and reporting
- Multilingual support (English primary)

## Technology Stack
- **Backend:** Effect TS + @effect/platform http
- **Frontend:** React with TanStack Router
- **Database:** PostgreSQL with double-entry accounting ledger
- **Real-time:** WebSocket-based communication
- **Observability:** OpenTelemetry instrumentation
- **Caching:** Redis-compatible solution
